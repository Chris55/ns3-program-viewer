# Rotary Speaker

## Stop Mode

        /***
         * Rotary Speaker Stop Mode:
         * Offset in file: 0x35 (bit7):
         *
         * Values:
         * 0 = enabled (Speed Stop), 1 = disabled (Speed Slow)
         */

## Speed

        /***
         * Rotary Speaker Speed
         * Offset in file: 0x34 (bit0):
         *
         * Values:
         * 0 = Slow/Stop, 1 = Fast
         */
